# 🚀 ResumeIQ--AI – Frontend

> Turning resumes into interactive AI experiences through contextual intelligence, modern UI, and Generative AI.

GenResume AI is a modern React application designed to help users understand, improve, and optimize their resumes using Large Language Models (LLMs). It provides an intuitive interface for AI-powered resume analysis, contextual resume conversations, ATS visualization, and personalized career recommendations.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/amansingh-gh/ResumeIQ--Frontend.git
```

Install dependencies

```bash
npm install
```

Create a `.env`

```env
VITE_API_URL=http://localhost:3000
```

Run locally

```bash
npm run dev
```
